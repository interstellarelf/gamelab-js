Gamelab.js: Gamepad-Input

October 28th 2020 Javascript | Gamelab-js Jordan E. Blake

Keyboard and mouse input will only do so much good for a game. Luckily, Gamelab-js has a solution for gamepad-input. This solution uses the HTMLGamepad Api, and works with most controllers, although its best suited for the XBox-PC Windows controller. Use the Gamepad Input functions to get callbacks on each button and control-stick, and directional-pad of your controller.
