October 27th 2020 Javascript | Gamelab-js Jordan E. Blake

We created the gamelab project with a single purpose: to display and manipulate graphic objects on-screen. 

                  
   //create a single sprite object
   var sprite = new gamelab.sprite('./assets/images/circle.png');

   sprite.onload(function(){
        this.position(100, 100);
        this.size(50, 50);
   });
