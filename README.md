#### Gamestack.js is a javascript library for 2D games.
-------------------------------------------------------------------

#### Features:
 -  js game-objects
 -  animations, sounds, gamepad-input
 -  events + behaviors
 -  framerate + memory tracking
 -  html + javascript examples + documentation
