Vector

Creates a Vector object with x, y, and z properties.

Vector-2D requires only x and y args --new Vector(10, 10) For Vector-3D, use x,y, and z --new Vector(10, 10, 10) Pass an existing Vector as the sole argument in order to copy that Vector to a new instance
