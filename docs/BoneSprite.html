BoneSprite

BoneSprite

Creates a new BoneSprite.

Gamelab.BoneSprite is a container for 2D Animations. -apply Sprite class to create a 2D game-object.

BoneSprites are a group of moving animations.

Constructor

new BoneSprite(src, scale) → {Sprite}

Parameters:
Name Type Default Description
src string

the srcPath for the image of the Sprite
scale number 1.0

the scale to be applied to size of each animation-frame
